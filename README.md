# VIPERUnitTestTemplate 

Unit Test Module Template For VIPER Architecture

To use the template, you only need to download this template, copy, and paste it to;

/Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/Library/Xcode/Templates/Project Templates/iOS/Application


- I do not have a full knowledge of using macros so there are a lot to improve here. 
- I invite you to contribute to this project to make it better. 
- inspired by @JuanpeCMiOS's [VIPER Template](https://github.com/Juanpe/Swift-VIPER-Module)
